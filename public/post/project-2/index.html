Project II: Chronic Kidney Disease Case Study

November 12, 2023

Date: January 2022 - March 2022

This is a course project on my STAT-630 Multivariate Analysis class, instructed by Dr. Matthew Schneider. The project was from the University of Virginia Darden College of Business. The project aims to perform multivariate analysis methods on the chronic kidney disease cases to understand the factors influencing whether a patient has such disease, and build predictive models to predictive CKD cases.

For the first part of the project, we used principal component analysis, factor analysis and cluster analysis to identify the key identifiers causing CKD.

To build models to predictive possible CKD patients, we used discriminant analysis and logistic regression to build classification models.
